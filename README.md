Guid for install macOS Mojave on Asus GL553VD 
----------------------------------------------
1- Only use clover folder for boot 
2- For install Trackpad -->
delete = AppleIntelLpssI2CController.kext + AppleIntelLpssI2C.kext in S/L/E
run kext utility
3- for install sound -->
install VoodooHDA = UEFI mode
